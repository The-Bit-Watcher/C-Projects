\doxysubsection{Detailed Description}
Represents a user in the chat system with command invocation capabilities. 

This class acts as the Invoker in the \mbox{\hyperlink{classCommand}{Command}} Pattern. It maintains a queue of commands and executes them in sequence when requested. \mbox{\hyperlink{classUsers}{Users}} can join multiple chat rooms and send messages to specific rooms.
